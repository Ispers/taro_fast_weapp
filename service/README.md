# taro_fast_weapp

#### 介绍
taro_fast（目前处于开发阶段，诚邀开发者加入，有想法的可以联系本人参与开发）是一套基于Taro+SpringBoot微信小程序前后端解决方案（同时也提供基于umi + SpringBoot管理员web端），接私活利器，使广大程序员可以基于此模版快速开发，专注于业务逻辑即可。
